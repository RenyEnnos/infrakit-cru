# Logs Detalhados

Esta pasta guarda registros longos de iterações importantes.

Use o formato:

```text
ITER-001-diagnostico-inicial.md
ITER-002-pesquisa-referencias.md
ITER-003-revisao-latex.md
```

Cada log deve conter:

- contexto da iteração;
- decisões tomadas;
- fontes consultadas, se houver;
- arquivos alterados;
- verificação realizada;
- riscos ou pendências.

O resumo da iteração também deve aparecer em `../REGISTRO_ITERACOES.md`.
